# go-chip

A chip-8 emulator. Special thanks to the technical documents [found here](http://devernay.free.fr/hacks/chip8/C8TECH10.HTM)

## Keyboard

Chip-8 uses a 16 key keypad for input which is mapped to the keyboard:

```text
---------       ---------
|1|2|3|C|   \   |1|2|3|4|
|4|5|6|D|  ==\  |Q|W|E|R|
|7|8|9|E|  ==/  |A|S|D|F|
|A|0|B|F|   /   |Z|X|C|V|
---------       ---------
```
